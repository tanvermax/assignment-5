What is the Document Object Model(DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content.

what is event delegation in the context of DOM ,and why is it usefull?

Event delegation is based event-bubbling method. It is an event-handling pattern that allows you to handle events at a higher level in the DOM tree other than the level where the event was first received.

How do you manipulate an elements attribute and style using DOM?

You can manipulate both an element's attributes and styles using the DOM in JavaScript.
1. Manipulating Attributes: To manipulate attributes, you can use methods like: getAttribute(): Retrieves the value of an attribute. setAttribute(): Sets a new value for an attribute. removeAttribute(): Removes an attribute from an element. hasAttribute(): Checks if an attribute exists.
